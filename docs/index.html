1. ブロック（Block）とは何か？

ブロックの基本概念

ブロックは、Rubyにおけるコードの一部分で、メソッドに渡すことができる匿名のコードの塊です。ブロックは、特定の処理をメソッドに委譲したり、繰り返し処理や条件分岐をカスタマイズする際に使用されます。

ブロックの構文

ブロックは主に以下の2つの構文で記述されます：

do...end構文

[1, 2, 3].each do |number|
  puts number
end

{...}構文

[1, 2, 3].each { |number| puts number }

両者は基本的に同じ意味を持ちますが、慣習としてdo...endは複数行のブロックに、{...}は一行のブロックに使われることが多いです。

2. yieldとは何か？

yieldは、メソッド内でブロックを呼び出すためのキーワードです。yieldを使うことで、メソッドに渡されたブロックを実行し、そのブロックに引数を渡すことができます。

yieldの基本的な使用方法

def greet
  yield "Hello"
end

greet do |message|
  puts message + ", world!"
end
# 出力: Hello, world!

この例では、greetメソッドがyieldを呼び出し、ブロックに"Hello"という引数を渡しています。ブロック内では、この引数を受け取り、出力しています。
